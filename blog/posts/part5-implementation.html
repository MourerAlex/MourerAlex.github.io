Implementation — From Formula to Code

Part 5: The g-formula and IPW estimators for separable effects in R, applied to the DES prostate cancer trial.

This post implements the estimators from Parts 4a and 4b on the DES prostate cancer trial. Event of interest ($Y$): prostate cancer death. Competing event ($D$): death from other causes. Treatment ($A$): DES ($A = 1$) vs placebo ($A = 0$). Baseline covariates only — the 2020 setting. The code follows Rojas-Saunero (SER 2023 Workshop) and Stensrud et al. (2020, 2021).

2. The person-time dataset

Every estimator operates on a person-time dataset: one row per patient per time interval. survSplit expands the one-row-per-subject data:

cut_times <- 0:59  # monthly intervals, 60 months follow-up

pt_df <- survSplit(
  data  = subject_df,
  cut   = cut_times,
  start = "t_start",
  end   = "event_time",
  event = "any_event"
)

Create separate event indicators, then apply the temporal ordering $(C_k, D_k, Y_k)$ from Part 4a Section 2: if censoring occurs, $D$ and $Y$ are unobserved; if $D$ occurs (uncensored), $Y$ is unobserved.

# Event type indicators
pt_df$y_event <- pt_df$any_event == 1 & pt_df$event_type == 1  # prostate death
pt_df$d_event <- pt_df$any_event == 1 & pt_df$event_type == 2  # other death
pt_df$c_event <- pt_df$any_event == 1 & pt_df$event_type == 0  # censored

# Temporal ordering: set unobserved outcomes to NA
pt_df$d_event[pt_df$c_event == 1] <- NA  # censored -> D unobserved
pt_df$y_event[pt_df$c_event == 1] <- NA  # censored -> Y unobserved
pt_df$y_event[pt_df$d_event == 1] <- NA  # D occurred -> Y unobserved

3. Pooled logistic regression

Every model is a pooled logistic regression: one glm on the stacked person-time data, with $k$ as a covariate. A polynomial in $k$ captures the baseline hazard shape. Interactions with treatment allow the treatment effect to change over time. To predict under a specific treatment value, override the treatment column inside the predict call:

# Predict Y hazard as if everyone received DES (A_y = 1)
predict(fit_y_haz, newdata = pt_df %>% mutate(A_y = 1), type = "response")

4. A note on censoring

Both estimators target the same quantity from Part 4a Section 2:

The target — cumulative incidence in the no-censoring world: $$\P(Y^{a_Y,\, a_D,\, \bar{c}=0}_{K+1} = 1)$$

The superscript $\bar{c} = 0$ means censoring has been intervened away. But the models are fitted on real data where some individuals are censored. The two estimators handle this differently.

The g-formula handles censoring through conditioning. The hazard models are conditioned on being uncensored: $\P(Y_{k+1} = 1 \mid \bar{C}_{k+1} = 0, \ldots)$. In the person-time data, censored individuals contribute rows up to their censoring time, then stop — glm drops NA rows automatically. The risk set at each time naturally excludes censored people. Under assumption E2 from Part 4a (censoring is ignorable given the covariates in the model), the fitted hazards on this observed risk set equal the counterfactual hazards under $\bar{c} = 0$ (Hernán & Robins 2020, Technical Point 21.10). No separate censoring model is needed. In practice, this means the hazard models must include all covariates that predict both the outcome and censoring — if they do, E2 is satisfied and the conditioning handles censoring automatically.

The IPW estimator handles censoring through weighting. It computes a weighted average among actually observed individuals in the $A = a_Y$ arm. Some of those individuals are censored — they disappear from the data. Without $W_C$ (Part 4a Section 5.5), the remaining uncensored individuals are a potentially biased subsample. $W_C$ upweights the uncensored to represent the full population under $\bar{c} = 0$. A censoring model is needed.

Both approaches require the same assumption (E2: censoring is ignorable given measured covariates). The mechanism differs: conditioning vs weighting.

5. The g-formula estimator

The g-formula plugs predicted hazards into the identification formula from Part 4a Section 4.8. No weights. Create hypothetical datasets, predict hazards under the intervention, and compute the cumulative incidence.

Target — the g-formula for separable effects (Part 4a, Section 4.8): $$\P(Y^{a_Y,\, a_D,\, \bar{c}=0}_{K+1} = 1) = \sum_{l} \sum_{s=0}^{K} \underbrace{\P(Y_{s+1} = 1 \mid \bar{C}_{s+1} = D_{s+1} = Y_s = 0,\; L = l,\; A = \color{teal}{a_Y})}_{\text{Block A: Y hazard from } \color{teal}{a_Y} \text{ arm}}$$ $$\times \prod_{j=0}^{s} \underbrace{\P(D_{j+1} = 0 \mid \bar{C}_{j+1} = D_j = Y_j = 0,\; L = l,\; A = \color{orange}{a_D})}_{\text{Block B: D event-free from } \color{orange}{a_D} \text{ arm}} \times \underbrace{\P(Y_j = 0 \mid \ldots,\; A = \color{teal}{a_Y})}_{\text{Block C: Y event-free}} \times \P(L = l)$$

5.1 — Fit the two hazard models

The $Y$-hazard model uses A_y. The $D$-hazard model uses A_d. In the observed data both equal $A$. They diverge at prediction time.

# Y hazard model: uses A_y as treatment variable
fit_y_haz <- glm(
  y_event ~ A_y * (k + I(k^2) + I(k^3)) + normal_act + age_cat + cv_hist + hemo_bin,
  data = pt_df, family = binomial()
)

# D hazard model: uses A_d as treatment variable
fit_d_haz <- glm(
  d_event ~ A_d * (k + I(k^2) + I(k^3)) + normal_act + age_cat + cv_hist + hemo_bin,
  data = pt_df, family = binomial()
)

Both models are fitted on the observed person-time data. Censored individuals contribute rows until their censoring time. The $\bar{C}_{k+1} = 0$ conditioning is implicit in the risk set (Section 4).

5.2 — Create cloned datasets

For each $(a_Y, a_D)$ combination, expand the baseline data over all time points with treatment set to the intervention values. The $\sum_l \ldots \P(L = l)$ in the formula becomes an average over individuals:

make_clone <- function(baseline_df, a_y, a_d, cut_times) {
  # baseline_df has one row per subject, guaranteeing each subject
  # appears exactly once per k in the clone
  n_subj <- nrow(baseline_df)
  K      <- length(cut_times)
  clone       <- baseline_df[rep(seq_len(n_subj), each = K), ]
  clone$k     <- rep(cut_times, n_subj)
  clone$A_y   <- a_y   # Y-component intervention value
  clone$A_d   <- a_d   # D-component intervention value
  return(clone)
}

# Three clones: observed DES, observed placebo, modified treatment
clone_11 <- make_clone(baseline_df, a_y = 1, a_d = 1, cut_times)  # DES
clone_00 <- make_clone(baseline_df, a_y = 0, a_d = 0, cut_times)  # Placebo
clone_10 <- make_clone(baseline_df, a_y = 1, a_d = 0, cut_times)  # Modified

Each clone has $n \times 60$ rows. In clone_10: A_y = 1 and A_d = 0. No censoring — every subject "lives" through all time points. This is the $\bar{c} = 0$ world.

5.3 — Predict hazards

Cross-arm prediction in clone_10 — the A_y/A_d trick: $$\hat{p}_Y(s+1) = \P(Y_{s+1} = 1 \mid \ldots,\; \underbrace{A = \color{teal}{1}}_{\texttt{A\_y} = 1}) \qquad \hat{p}_D(s+1) = \P(D_{s+1} = 1 \mid \ldots,\; \underbrace{A = \color{orange}{0}}_{\texttt{A\_d} = 0})$$ 
predict_hazards <- function(clone, fit_y, fit_d) {
  # Y hazard: fit_y_haz looks for A_y in the data
  clone$haz_y <- predict(fit_y, newdata = clone, type = "response")
  # D hazard: fit_d_haz looks for A_d in the data
  clone$haz_d <- predict(fit_d, newdata = clone, type = "response")
  # Joint event-free survival at each interval
  clone$surv  <- (1 - clone$haz_y) * (1 - clone$haz_d)
  return(clone)
}

clone_11 <- predict_hazards(clone_11, fit_y_haz, fit_d_haz)
clone_00 <- predict_hazards(clone_00, fit_y_haz, fit_d_haz)
clone_10 <- predict_hazards(clone_10, fit_y_haz, fit_d_haz)

In clone_10: fit_y_haz sees A_y = 1 → $Y$ hazard under DES. fit_d_haz sees A_d = 0 → $D$ hazard under placebo. The g-formula mixes hazards from two arms.

5.4 — Compute cumulative incidence

Iterative computation of the cumulative incidence (Part 4a, Section 4.8): $$S_i(0) = 1$$ $$S_i(s) = S_i(s-1) \times \underbrace{(1 - \hat{p}_{Y,i}(s))}_{\texttt{1 - haz\_y}} \times \underbrace{(1 - \hat{p}_{D,i}(s))}_{\texttt{1 - haz\_d}}$$ $$\text{inc}_i(s+1) = \underbrace{\hat{p}_{Y,i}(s+1)}_{\text{Block A}} \times \underbrace{(1 - \hat{p}_{D,i}(s+1))}_{\text{Block B}} \times S_i(s)$$ $$\hat{\nu}_{a_Y, a_D} = \frac{1}{n}\sum_{i=1}^{n} \sum_{s=0}^{K} \text{inc}_i(s+1)$$ 
compute_cum_inc <- function(clone_df) {
  result <- clone_df %>%
    group_by(id) %>%
    mutate(
      # S_i(s): cumulative event-free survival, built iteratively
      cum_surv     = cumprod(surv),
      # S_i(s-1): survival through PREVIOUS interval. At k=0, S(-1) = 1
      lag_cum_surv = lag(cum_surv, default = 1),
      # Incidence increment: Y hazard * D event-free * prior survival
      inc          = haz_y * (1 - haz_d) * lag_cum_surv
    ) %>%
    ungroup() %>%
    # Average across subjects at each time (standardisation over L)
    group_by(k) %>%
    summarise(mean_inc = mean(inc), .groups = "drop") %>%
    arrange(k) %>%
    # Cumulate over time to get the cumulative incidence curve
    mutate(cum_inc = cumsum(mean_inc))

  return(result)
}

cum_inc_11 <- compute_cum_inc(clone_11)  # nu(1,1): DES
cum_inc_00 <- compute_cum_inc(clone_00)  # nu(0,0): Placebo
cum_inc_10 <- compute_cum_inc(clone_10)  # nu(1,0): Modified treatment

6. The IPW estimator

The g-formula requires both hazard models to be correct. IPW Representation 1 needs only the $D$ hazard model (plus censoring). It stands in the $A = \color{teal}{a_Y}$ arm. In that arm, two of the three g-formula blocks are already at the right value ($Y$ hazard and $Y$ event-free at $\color{teal}{a_Y}$). Only the $D$ event-free block needs fixing — $W_D$ swaps it from $\color{teal}{a_Y}$ to $\color{orange}{a_D}$.

Target — IPW Representation 1 (Part 4a, Section 5.6): $$\nu_{a_Y, a_D, k} = \sum_{s=0}^{k} \E\Big[\underbrace{W'_{C,s}}_{\texttt{w\_cens}} \cdot \underbrace{W_{D,s}}_{\texttt{w\_d}} \cdot Y_{s+1}(1 - Y_s)(1 - D_{s+1}) \;\Big|\; A = \color{teal}{a_Y}\Big]$$ $$W_{D,s} = \prod_{j=0}^{s} \frac{\P(D_{j+1} = 0 \mid \ldots,\; A = \color{orange}{a_D})}{\P(D_{j+1} = 0 \mid \ldots,\; A = \color{teal}{a_Y})} \qquad W'_{C,s} = \frac{1}{\prod_{j=0}^{s} \P(C_{j+1} = 0 \mid \ldots,\; A = \color{teal}{a_Y})}$$

6.1 — Fit models

The $D$ hazard model (same as in the g-formula). Plus a censoring model — Section 4 explains why IPW needs it. The censoring model uses the observed treatment $A$: since $W_C$ is only applied within the $A = a_Y$ arm, predicting at the observed value is already predicting at $a_Y$.

# D hazard model (same as g-formula, uses A_d)
fit_d_haz <- glm(
  d_event ~ A_d * (k + I(k^2) + I(k^3)) + normal_act + age_cat + cv_hist + hemo_bin,
  data = pt_df, family = binomial()
)

# Censoring model (uses observed A — see Section 4)
fit_cens <- glm(
  c_event ~ A * (k + I(k^2) + I(k^3)) + normal_act + age_cat + cv_hist + hemo_bin,
  data = pt_df, family = binomial()
)

6.2 — Predict D survival at both arms

Numerator and denominator of $W_D$: $$\texttt{surv\_d\_ad} = \P(D_{j+1} = 0 \mid \ldots,\; A = \color{orange}{a_D}) \qquad \texttt{surv\_d\_ay} = \P(D_{j+1} = 0 \mid \ldots,\; A = \color{teal}{a_Y})$$ 
# D survival under a_D = 0: numerator of W_D (the rate we WANT)
pt_df$surv_d_ad <- 1 - predict(
  fit_d_haz,
  newdata = pt_df %>% mutate(A_d = 0),  # predict at placebo D rate
  type = "response"
)

# D survival under a_Y = 1: denominator of W_D (the rate we CANCEL)
pt_df$surv_d_ay <- 1 - predict(
  fit_d_haz,
  newdata = pt_df %>% mutate(A_d = 1),  # predict at DES D rate
  type = "response"
)

# Censoring survival at observed treatment (no override needed)
pt_df$surv_cens <- 1 - predict(fit_cens, newdata = pt_df, type = "response")

6.3 — Compute weights

pt_df <- pt_df %>%
  group_by(id) %>%  # cumulative products must be per subject
  mutate(
    # Cumulative D survival at each arm
    cum_surv_d_ad = cumprod(surv_d_ad),  # product through time k
    cum_surv_d_ay = cumprod(surv_d_ay),
    cum_surv_cens = cumprod(surv_cens),
    # W_D: swap D distribution from a_Y rate to a_D rate
    w_d     = cum_surv_d_ad / cum_surv_d_ay,
    # W_C: upweight uncensored to represent censored
    w_cens  = 1 / cum_surv_cens,
    # Total weight
    w_total = w_d * w_cens
  ) %>%
  ungroup()

$W_D$ swaps the $D$ distribution: the denominator cancels the DES cardiovascular death rate in the $\color{teal}{a_Y} = 1$ arm; the numerator inserts the placebo rate. In the 2021 setting with time-varying covariates, a third weight $W_{L_D}$ (Part 4b Section 5) would multiply in. For baseline covariates only, $W_{L_D} = 1$.

6.4 — The weighted estimator

Among individuals in the $A = \color{teal}{a_Y}$ arm: at each time $k$, sum the weights of those who experience $Y$, divided by the number of subjects in the arm at baseline. Cumulate:

estimate_sep_eff <- function(pt_df, a_y, max_time) {
  # Keep only individuals in the a_Y arm
  ay_df <- pt_df[pt_df$A == a_y, ]
  # Denominator: subjects at baseline, not current risk set
  n_ay  <- sum(ay_df$k == 0)

  times     <- seq_len(max_time)
  event_vec <- numeric(max_time)

  for (s in times) {
    # Risk set at time s: uncensored, D-free
    riskset_s <- ay_df[ay_df$k == s & ay_df$c_event == 0 & ay_df$d_event == 0, ]
    # Subjects who experienced Y at time s
    eventset_s <- riskset_s[riskset_s$y_event == 1, ]
    # Weighted incidence increment
    event_vec[s] <- sum(eventset_s$w_total) / n_ay
  }

  # Cumulate over time
  return(data.frame(k = times, cum_inc = cumsum(event_vec)))
}

# Observed arms (a_Y = a_D): no D swap needed, W_D = 1
# Override w_total to use only censoring weights
cum_inc_11_ipw <- estimate_sep_eff(
  pt_df %>% mutate(w_total = w_cens),
  a_y = 1, max_time = 59
)
cum_inc_00_ipw <- estimate_sep_eff(
  pt_df %>% mutate(w_total = w_cens),
  a_y = 0, max_time = 59
)

# Cross-arm (a_Y=1, a_D=0): full separable effect weight
cum_inc_10_ipw <- estimate_sep_eff(pt_df, a_y = 1, max_time = 59)

The g-formula and IPW curves should be close if both sets of models are correctly specified. Discrepancies suggest model misspecification. Computing both is the strongest practical check. Weight diagnostics and bootstrap confidence intervals are standard (Hernán & Robins 2020, Chapters 12 and 17).

7. Causal contrasts

The g-formula and IPW each produce cumulative incidence curves $\hat{\nu}(a_Y, a_D, k)$ — the risk of $Y$ by month $k$ under a specific treatment combination. The causal questions are answered by contrasts between these curves:

Risk differences at time $K$ (Part 2, Section 5): $$\text{Total effect} = \hat{\nu}(1,1,K) - \hat{\nu}(0,0,K)$$ $$\text{Separable indirect effect } (A_D) = \hat{\nu}(1,1,K) - \hat{\nu}(1,0,K)$$ $$\text{Separable direct effect } (A_Y) = \text{Total} - \text{Indirect}$$ 
# Risk differences at month 60 (using g-formula estimates)
K <- 60
total_eff    <- cum_inc_11$cum_inc[K] - cum_inc_00$cum_inc[K]
sep_indirect <- cum_inc_11$cum_inc[K] - cum_inc_10$cum_inc[K]
sep_direct   <- total_eff - sep_indirect

data.frame(
  estimand = c("Total effect", "Separable indirect (A_D)", "Separable direct (A_Y)"),
  risk_diff = round(c(total_eff, sep_indirect, sep_direct), 4)
)

The separable indirect effect ($A_D$) captures how much of DES’s effect on prostate cancer death operates through its effect on cardiovascular death. The separable direct effect ($A_Y$) captures the effect through testosterone suppression alone. These risk difference curves can also be plotted over time — the gap between the DES curve and the modified curve at each month $k$ traces the separable indirect effect.

8. Summary

Models neededExpectation overCensoring handling
G-formula$Y$ hazard + $D$ hazardBoth armsConditioning (implicit in risk set)

The g-formula uses all information (both hazards) but requires both to be correct. IPW avoids the $Y$ hazard model entirely. A mirror IPW Representation 2 (Part 4a Section 6) avoids the $D$ model instead. Computing all estimators and comparing is the strongest consistency check.

This post covers the baseline-covariates-only setting (Stensrud et al. 2020). When time-varying covariates are present (Stensrud et al. 2021), the IPW estimator acquires an additional weight $W_{L_D}$ via the Bayes’ rule trick (Part 4b Section 5), and the g-formula requires Monte Carlo simulation or the ICE representation. These extensions, along with sensitivity analysis for violations of the isolation conditions, will be covered in a future post.
