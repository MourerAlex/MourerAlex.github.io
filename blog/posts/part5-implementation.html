Implementation — From Formula to Code

Part 5: The g-formula and IPW estimators for separable effects, implemented step by step in R on the DES prostate cancer trial.

This post implements the estimators from Parts 4a and 4b on the DES prostate cancer trial. Event of interest ($Y$): prostate cancer death. Competing event ($D$): death from other causes. Treatment ($A$): DES ($A = 1$) vs placebo ($A = 0$). Baseline covariates only — the 2020 setting. The code follows the implementations by Rojas-Saunero (SER 2023 Workshop) and Stensrud et al. (2020, 2021).

2 — Person-time expansion

We expand to one row per subject per time interval using a single call to survSplit. We set event_indicator = 1L for all subjects, including censored ones. survSplit uses this to flag each subject's terminal row.

subject_df <- subject_df %>%
  mutate(event_indicator = 1L)

cut_times <- 1:75

pt_df <- survSplit(
  data  = subject_df,
  cut   = cut_times,
  start = "tstart",
  end   = "event_time",
  event = "event_indicator"
)

pt_df <- pt_df %>%
  rename(tstop = event_time) %>%    # interval boundaries [tstart, tstop)
  mutate(k = tstart)                # k = interval index (month 0, 1, 2, ...)

Derive the three event indicators from event_indicator + event_type:

pt_df <- pt_df %>%
  mutate(
    y_event = as.integer(event_indicator == 1 & event_type == 1),  # prostate death
    d_event = as.integer(event_indicator == 1 & event_type == 2),  # other death
    c_event = as.integer(event_indicator == 1 & event_type == 0)   # censored/alive
  )

Temporal ordering

The ordering within each interval is $C_k$, then $D_k$, then $Y_k$. If censoring happens, $D$ and $Y$ are unobserved. If $D$ happens, $Y$ is unobserved. We encode this by setting the unobserved indicators to NA. When glm encounters NA in the outcome, it drops that row — which is exactly the correct risk set for each model.

pt_df <- pt_df %>%
  mutate(
    d_event = ifelse(c_event == 1, NA, d_event),    # censored → D unobserved
    y_event = ifelse(c_event == 1, NA, y_event),    # censored → Y unobserved
    y_event = ifelse(d_event == 1, NA, y_event),    # D occurred → Y unobserved
    A_y = A,                                         # Y-hazard model sees A_y
    A_d = A                                          # D-hazard model sees A_d
  )

baseline_df <- pt_df %>% filter(k == 0)   # one row per subject
n <- n_distinct(pt_df$id)

A_y and A_d are copies of A. In the observed data they are identical. They diverge in the cloned datasets (Section 4), where the cross-arm trick sets them to different values.

3 — A note on censoring

The g-formula and IPW handle censoring differently:

G-formula: no separate censoring model. Censoring is handled by conditioning — the covariates that predict censoring are included in the hazard models. The g-formula integrates over the full covariate history, accounting for informative censoring as long as the relevant predictors are in the models. The cloned datasets have no censoring: every subject "lives" through all time points. This is the $\bar{c} = 0$ world.

IPW: requires an explicit censoring model. IPW operates on observed individuals, and censored subjects are missing from the later risk sets. The censoring weights $W_C$ upweight the uncensored subjects to represent those who were lost.

4 — G-formula plug-in estimator

The target is a counterfactual quantity: $\P(Y^{a_Y,\, a_D,\, \bar{c}=0}_{K+1} = 1)$ — the risk of prostate cancer death had treatment components been set to $a_Y$ and $a_D$ and nobody been censored. This quantity cannot be observed directly.

The identification step (Part 4a, Section 4.8) showed that under the dismissible component conditions, exchangeability, consistency, and positivity, this counterfactual equals an expression involving only observed-data conditional probabilities — the cause-specific hazards at each time, conditioned on covariates and treatment:

The g-formula for separable effects (Part 4a, Section 4.8 / Stensrud 2020, eq. 8/9): $$\nu(a_Y, a_D) = \sum_l \left[ \sum_{s=0}^{K} \underbrace{\P(Y_{s+1}\!=\!1 \mid D_{s+1}\!=\!Y_s\!=\!0,\; A\!=\!\color{teal}{a_Y},\; L\!=\!l)}_{\text{Block A}} \prod_{j=0}^{s} \left( \underbrace{\P(D_{j+1}\!=\!0 \mid D_j\!=\!Y_j\!=\!0,\; A\!=\!\color{orange}{a_D},\; L\!=\!l)}_{\text{Block B}} \times \underbrace{\P(Y_j\!=\!0 \mid D_j\!=\!Y_{j-1}\!=\!0,\; A\!=\!\color{teal}{a_Y},\; L\!=\!l)}_{\text{Block C}} \right) \right] \P(L\!=\!l)$$

The entire Part 4a derivation was to reach this expression — converting an unobservable causal quantity into a function of observable conditional probabilities. Block A: $Y$-hazard from the $\color{teal}{a_Y}$ arm. Block B: $D$-free probability from the $\color{orange}{a_D}$ arm. Block C: $Y$-free probability from the $\color{teal}{a_Y}$ arm. The key: Blocks A and C are evaluated at $A = \color{teal}{a_Y}$, Block B at $A = \color{orange}{a_D}$ — different arms in the cross-arm regime $(1, 0)$.

We write $\hat{h}_Y$ and $\hat{h}_D$ for the discrete hazards — these are conditional probabilities of the event at a given interval, given survival to that interval. They are the quantities that predict(..., type = "response") returns from the pooled logistic models. Replacing each conditional probability with its estimate $\hat{h}$ and using the empirical average $\frac{1}{n}\sum_i$ in place of $\sum_l \ldots \P(L=l)$:

Implementation form (maps directly to the code): $$\hat{\nu}(a_Y, a_D) = \frac{1}{n}\sum_{i=1}^{n} \sum_{s=0}^{K} \;\underbrace{\hat{h}_Y(s\!+\!1 \mid l_i,\, \color{teal}{a_Y})}_{\text{Block A}} \;\times\; \underbrace{(1-\hat{h}_D(s\!+\!1 \mid l_i,\, \color{orange}{a_D})) \;\prod_{j=0}^{s-1}(1-\hat{h}_D(j\!+\!1 \mid l_i,\, \color{orange}{a_D}))}_{\text{Block B}} \;\times\; \underbrace{\prod_{j=0}^{s}(1-\hat{h}_Y(j \mid l_i,\, \color{teal}{a_Y}))}_{\text{Block C}}$$

The estimation step replaces each true conditional hazard with an estimate from a pooled logistic regression. "Pooled" means all person-time records from all individuals at all times are stacked into one dataset, and one glm is fitted on the whole thing. Time enters as a covariate, so the model can produce a different hazard at each month. Concretely, for the $Y$-hazard:

The pooled logistic model for the $Y$-hazard: $$\text{logit}\;\hat{h}_Y(k \mid l_i, a_Y) = \beta_0 + \beta_1 a_Y + \beta_2 k + \beta_3 k^2 + \beta_4 k^3 + \beta_5 \,\texttt{normal\_act}_i + \cdots$$

All coefficients $(\beta_0, \ldots)$ are estimated simultaneously from the stacked person-time data. An individual followed for 20 months contributes 20 rows. An individual followed for 60 months contributes 60 rows. The polynomial in $k$ is what allows the model to produce a different hazard at each month — it learns the shape of the baseline hazard from the pooled data. The treatment effect ($\beta_1$) is constant over time. Once the model is fitted, predict evaluates it at any combination of $(k, a_Y, l_i)$.

This is how the identification formula becomes computable: predict the $Y$-hazard at $a_Y$ and the $D$-hazard at $a_D$, for each individual at each time. The cloned datasets (below) are the mechanism — by setting A_y and A_d to different values, each model sees the treatment value prescribed by the identification formula.

4.1 — Fit hazard models

fit_y_haz <- glm(
  y_event ~ A_y + k + I(k^2) + I(k^3) + normal_act + age_cat + cv_hist + hemo_bin,
  data = pt_df, family = binomial()
)

fit_d_haz <- glm(
  d_event ~ A_d + k + I(k^2) + I(k^3) + normal_act + age_cat + cv_hist + hemo_bin,
  data = pt_df, family = binomial()
)

Both models are fitted on the observed person-time data. fit_y_haz uses A_y; fit_d_haz uses A_d. Censored individuals contribute rows until their censoring time, then stop. The $\bar{C}_{k+1} = 0$ conditioning is implicit in the risk set.

4.2 — Create cloned datasets

For each $(a_Y, a_D)$ regime, expand the baseline data over all time points with treatment set to the intervention values. The $\sum_l f(l)\,\P(L = l)$ standardisation becomes $(1/n)\sum_i f(l_i)$: each subject carries their own covariates, the empirical average is the standardisation.

make_clone <- function(baseline_df, a_y, a_d, times) {
  n_subj <- nrow(baseline_df)
  K      <- length(times)
  clone       <- baseline_df[rep(seq_len(n_subj), each = K), ]
  clone$k     <- rep(times, n_subj)
  clone$A_y   <- a_y   # Y-component intervention value
  clone$A_d   <- a_d   # D-component intervention value
  return(clone)
}

time_points <- c(0, cut_times)   # all k values: 0, 1, ..., 75

clone_11 <- make_clone(baseline_df, a_y = 1, a_d = 1, time_points)  # DES
clone_00 <- make_clone(baseline_df, a_y = 0, a_d = 0, time_points)  # Placebo
clone_10 <- make_clone(baseline_df, a_y = 1, a_d = 0, time_points)  # Modified

In clone_10: A_y = 1 and A_d = 0. No censoring — every subject "lives" through all time points.

4.3 — Predict hazards (the cross-arm trick)

Cross-arm prediction in clone_10: $$\hat{h}_Y(s\!+\!1) = \P(Y_{s+1} = 1 \mid \ldots,\; \underbrace{A = \color{teal}{1}}_{\texttt{A\_y} = 1}) \qquad \hat{h}_D(s\!+\!1) = \P(D_{s+1} = 1 \mid \ldots,\; \underbrace{A = \color{orange}{0}}_{\texttt{A\_d} = 0})$$ 
predict_hazards <- function(clone, fit_y, fit_d) {
  clone$haz_y <- predict(fit_y, newdata = clone, type = "response")  # fit_y sees A_y
  clone$haz_d <- predict(fit_d, newdata = clone, type = "response")  # fit_d sees A_d
  # Joint event-free probability within ONE interval (not cumulative).
  clone$surv  <- (1 - clone$haz_y) * (1 - clone$haz_d)
  return(clone)
}

clone_11 <- predict_hazards(clone_11, fit_y_haz, fit_d_haz)
clone_00 <- predict_hazards(clone_00, fit_y_haz, fit_d_haz)
clone_10 <- predict_hazards(clone_10, fit_y_haz, fit_d_haz)

In clone_10: fit_y_haz sees A_y = 1 → $Y$ hazard under DES. fit_d_haz sees A_d = 0 → $D$ hazard under placebo. This is the cross-arm prediction from Part 4a.

4.4 — Compute cumulative incidence

From equation to code — the math traceback

Define shorthand for subject $i$ with covariates $l_i$:

  • $\hat{h}_Y(s \mid i) \equiv \hat{h}_Y(s \mid l_i, a_Y)$
  • $\hat{h}_D(s \mid i) \equiv \hat{h}_D(s \mid l_i, a_D)$

The product $S_i(s) = \prod_{j=0}^{s}(1 - \hat{h}_Y(j \mid i))(1 - \hat{h}_D(j \mid i))$ is the joint event-free survival through interval $s$. In the paper, $j$ runs from 0 to $s$, so $D$-survival at $s+1$ is NOT in the product — it's in the conditioning of the $Y$-hazard (Block B).

Temporal ordering within interval $s+1$: at interval $s+1$, $D$ is resolved first, then $Y$. So the incidence increment at interval $s+1$ is:

  • Survive all prior intervals: $S_i(s)$
  • At interval $s+1$: survive $D$ (prob $1 - \hat{h}_D(s\!+\!1 \mid i)$), then experience $Y$ (prob $\hat{h}_Y(s\!+\!1 \mid i)$)

This maps directly to:

inc = haz_y * (1 - haz_d) * lag_cum_surv

where lag_cum_surv $= S_i(s)$ via lag(cumprod(surv), default = 1), and surv $= (1 - \hat{h}_Y)(1 - \hat{h}_D)$.

The standardisation $\sum_l f(l)\P(L = l)$ becomes the sample average $\frac{1}{n}\sum_i f(l_i)$. Each subject $i$ provides one draw from the covariate distribution — the empirical distribution assigns weight $1/n$ to each observed $l_i$. This is why we compute per-subject first (each has a different hazard trajectory due to different covariates), then average across subjects:

$$\nu = \sum_{s} \underbrace{\frac{1}{n}\sum_i \text{inc}_i(s+1)}_{\texttt{mean(inc) grouped by k}} \quad\longrightarrow\quad \texttt{cumsum(mean\_inc)}$$
Iterative form (maps directly to the code): $$S_i(0) = 1$$ $$S_i(s) = S_i(s\!-\!1) \;\times\; \bigl(1 - \hat{h}_Y(s \mid l_i)\bigr)\;\bigl(1 - \hat{h}_D(s \mid l_i)\bigr)$$ $$\text{inc}_i(s\!+\!1) = \hat{h}_Y(s\!+\!1 \mid l_i) \;\times\; \bigl(1 - \hat{h}_D(s\!+\!1 \mid l_i)\bigr) \;\times\; S_i(s)$$ $$\nu = \sum_{s} \frac{1}{n}\sum_i \text{inc}_i(s\!+\!1)$$

This is the parametric g-formula, not Aalen-Johansen. We must compute per-subject first, then average: $\text{cumprod}(\text{mean}(\text{surv})) \neq \text{mean}(\text{cumprod}(\text{surv}))$ because each subject has different covariates → different hazard trajectories → different survival paths.

compute_cum_inc <- function(clone_df) {
  clone_df %>%
    group_by(id) %>%
    arrange(k) %>%
    mutate(
      cum_surv     = cumprod(surv),                # S_i(s): cumulative event-free survival
      lag_cum_surv = lag(cum_surv, default = 1),    # S_i(s-1); default=1 implies S_i(0)=1 at k=0
      inc          = haz_y * (1 - haz_d) * lag_cum_surv   # subdensity increment
    ) %>%
    ungroup() %>%
    group_by(k) %>%
    summarise(mean_inc = mean(inc), .groups = "drop") %>%
    arrange(k) %>%
    mutate(cum_inc = cumsum(mean_inc))
}

cum_inc_11 <- compute_cum_inc(clone_11)   # ν(1,1): DES
cum_inc_00 <- compute_cum_inc(clone_00)   # ν(0,0): Placebo
cum_inc_10 <- compute_cum_inc(clone_10)   # ν(1,0): Modified

5 — IPW estimator

IPW Representation 1 (Part 4a Section 5.6): stand in the $A = a_Y$ arm. $Y$ hazards and $Y$ events are already correct in that arm. $W_D$ swaps the $D$ distribution from $a_Y$ to $a_D$. $W_C$ corrects for censoring. No $Y$ hazard model needed — just observed $Y$ events, reweighted. Trade-off: doesn't need the $Y$ model to be correct, but needs the $D$ and censoring models to be correct.

5.1 — Fit censoring model

The censoring model uses the observed treatment $A$, not an overridden $a_Y$. This is because $W_C$ is only applied to individuals in the $A = a_Y$ arm (the estimate_sep_eff function filters to that arm). For those individuals, $A = a_Y$ is already true. Predicting at the observed value is predicting at $a_Y$.

fit_cens <- glm(
  c_event ~ A + k + I(k^2) + I(k^3) + normal_act + age_cat + cv_hist + hemo_bin,
  data = pt_df, family = binomial()
)

5.2 — Predict hazards at both arms

$W_D$ weight (Stensrud 2020, eq. 11): $$W_D(s) = \frac{\prod_{j=0}^{s}\bigl(1 - \hat{h}_D(j \mid l,\, \color{orange}{a_D})\bigr)}{\prod_{j=0}^{s}\bigl(1 - \hat{h}_D(j \mid l,\, \color{teal}{a_Y})\bigr)}$$

Numerator: cumulative $D$-free survival under the rate we want ($a_D$). Denominator: cumulative $D$-free survival under the rate we cancel ($a_Y$).
In code: $\hat{h}_D(j \mid l, a_D)$ = haz_d_ad, $\hat{h}_D(j \mid l, a_Y)$ = haz_d_ay. Then net_surv_d_ad = cumprod(1 - haz_d_ad) and net_surv_d_ay = cumprod(1 - haz_d_ay).
$W_D > 1$ when $D$ is more likely under $a_Y$ than $a_D$: surviving $D$ in the $a_Y$ arm is "too rare", so we upweight those who made it. 

pt_df <- pt_df %>%
  mutate(
    haz_d_ad = predict(fit_d_haz,                                # D-hazard at a_D (rate we WANT)
                       newdata = pt_df %>% mutate(A_d = a_d),
                       type = "response"),
    haz_d_ay = predict(fit_d_haz,                                # D-hazard at a_Y (rate to CANCEL)
                       newdata = pt_df %>% mutate(A_d = a_y),
                       type = "response"),
    haz_cens = predict(fit_cens, newdata = pt_df,                # censoring hazard
                       type = "response")
  )

5.3 — Compute weights

The cumulative products below are net survivals — survival under hypothetical elimination of the other event type (the same quantity as the "marginal cumulative incidence" from Part 1, but here on the survival scale). They are used purely as computational building blocks for the weights, not as estimands.

pt_df <- pt_df %>%
  group_by(id) %>%               # cumprod MUST be per-subject
  arrange(k) %>%
  mutate(
    net_surv_d_ad = cumprod(1 - haz_d_ad),     # net D-survival under a_D (D-free, eliminating Y)
    net_surv_d_ay = cumprod(1 - haz_d_ay),     # net D-survival under a_Y (D-free, eliminating Y)
    net_surv_cens = cumprod(1 - haz_cens),     # net censoring survival
    w_d     = net_surv_d_ad / net_surv_d_ay,    # swap D distribution
    w_cens  = 1 / net_surv_cens,                # correct for censoring
    w_total = w_d * w_cens                      # combined weight
  ) %>%
  ungroup()

5.4 — Weighted estimator

IPW cumulative incidence (Part 4a eq. 11 / Young et al. 2020 eq. 38): $$\hat{\nu}(a_Y, a_D) = \sum_{s=0}^{K} \frac{1}{n_{a_Y}} \sum_{i:\, A_i = a_Y} W_D(s) \cdot W_C(s) \cdot \mathbb{1}\{Y_{i,s+1} = 1,\; D_{i,s+1} = 0,\; C_{i,s+1} = 0\}$$

Divide by $n$ at baseline, not current risk set. This is Horvitz-Thompson: the weights already correct for attrition.

estimate_sep_eff <- function(pt_df, a_y_val, max_k) {
  # Keep only individuals in the a_Y arm
  ay_df <- pt_df %>% filter(A == a_y_val)
  # Denominator: number of subjects at baseline, NOT current risk set
  n_ay  <- sum(ay_df$k == 0)
  times <- sort(unique(ay_df$k))
  times <- times[times > 0 & times <= max_k]

  event_vec <- numeric(length(times))
  for (i in seq_along(times)) {
    s <- times[i]
    # Risk set: alive, uncensored, D-free at time s
    riskset_s  <- ay_df %>%
      filter(k == s, c_event == 0, d_event == 0)
    # Y events at time s
    eventset_s <- riskset_s %>% filter(y_event == 1)
    # Weighted incidence increment
    event_vec[i] <- sum(eventset_s$w_total) / n_ay
  }
  return(data.frame(k = times, cum_inc = cumsum(event_vec)))
}

For the observed arms ($a_Y = a_D$), $W_D = 1$ and only censoring weights apply:

max_k <- max(pt_df$k)
